# Точка работы

Курсовой веб-проект для поиска и бронирования коворкингов.

Проект выполнен на базовом стеке фронтенд-технологий:

- HTML;
- CSS;
- JavaScript.

## Описание

Веб-интерфейс позволяет просматривать доступные рабочие пространства, искать коворкинги по названию или району, фильтровать их по типу и количеству гостей, а затем оформлять бронирование на выбранную дату и время.

## Основные функции

- отображение карточек коворкингов;
- поиск по названию, адресу и району;
- фильтрация по типу пространства;
- фильтрация по удобствам;
- фильтрация по количеству гостей;
- добавление собственных коворкингов;
- редактирование и удаление добавленных коворкингов;
- добавление коворкингов в избранное;
- отдельная вкладка с избранными пространствами;
- просмотр подробного описания в модальном окне;
- выбор даты, времени и длительности бронирования;
- автоматический расчет итоговой стоимости;
- проверка вместимости пространства;
- создание и отмена бронирований;
- редактирование даты, времени, длительности и количества гостей;
- сохранение бронирований в `localStorage`;
- переключение светлой и темной темы;
- адаптивное отображение на мобильных устройствах.

## Назначение проекта

Проект демонстрирует базовые навыки создания клиентского веб-приложения: семантическую HTML-разметку, адаптивную верстку, работу с массивами объектов, обработку событий, динамическое изменение DOM и хранение данных на стороне браузера.
